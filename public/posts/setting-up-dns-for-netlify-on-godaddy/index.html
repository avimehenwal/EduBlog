How to set up DNS for Netlify on Godaddy

15 Feb 2019
myLearning deployment dns godaddy
@Avi Mehenwal

Easy way to setup and configure DNS on godaddy to be used by netlify for custome domain setting.

  1. Once your website is deployed and ready on netlify, copy the name netlify server name (url where it is hosted by netlify). It should be something like youthful-villani-10bc56.netlify.com
  2. Log in and open godaddy console -> Manage DNS for your selection and then add the following A and CNAME records
Type	Name	Value
A       @      104.198.14.52
CNAME	www    youthful-villani-10bc56.netlify.com
  1. Go back to netlify console -> Domain Setting, and then add you domain name to custome domain

Save everything and wait for changes to reflect. You are done!

Enjoy your new website
